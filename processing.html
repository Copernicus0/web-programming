Ce este processing?

ProcessingJS( sau P5JS) este o bibliotecă Javascript ce permite desenarea programată (prin intermediul unor funcții și instrucțiuni) pe o pânză de desenare.

ProcessingJS a fost gândit să fie ușor de folosit, mai ales pentru începători, astfel că este un mediu prin care oricine poate învăța programare. Cu Processing se pot desena pe pânză diverse figuri (cercuri, elipse, dreptunghiuri, etc) , se poate scrie text și, prin desenarea succesivă a pânzei se pot obține animații.

Ce este o panza?

Pânza de desenare (HTML Canvas) este un marcaj nou în HTML5.

Practic, o pânză de desenare este un dreptunghi cu dimensiuni cunoscute (stabilite la definirea pânzei), alcătuit din pixeli,fiecare pixel putând fi colorat independent, prin intermediul coordonatelor sale.

Coordonatele sunt exprimate printr-un sistem de coordonate în care originea (punctul de coordonate (0,0)) se află în colțul stânga-sus al pânzei, abscisa (coordonata x) se consideră pe orizontală (pe lățimea pânzei, width) iar ordonata (coordonata Y) se consideră pe verticală (pe înălțimea pânzei, height).

Astfel poti creea jocuri si/sau animatii.

Cum fac sa funcționeze?

Trebuie sa punem in eticheta <head> urmatorul script:

<script src="https://cdnjs.cloudflare.com/ajax/libs/processing.js/1.6.0/processing.min.js"></script>

Câteva funcții esențiale

ellipse(x,y,width,height);→ generează o elipsă la coordonatele x și y,iar width și height reprezintă lățimea respectiv înalțimea elipsei.

rect(x,y,width,height);→ generează un dreptunghi.

fill(roșu,verde,albastru);→ umple un element cu culoare(vezi pagina CSS)

text("aici pui textul",x,y);→ permite afișarea textului la coordonatele x si y,

noStroke();→ nu mai pune contur obiectului.

triangle(x1,y1,x2,y2,x3,y3);→ face un triunghi.

void setup(){conținut}→ execută celelalte funcții(conținutul) o dată.

void draw(){conținut}→ execută conținutul de mai multe ori.

Funcțiile setup() și draw() sunt foarte importante deoarece fară ele nu se poate afișa nimic.

Sintaxa:

<script type="application/processing">
int eyes=40;
int head=200;
int mouth=40;
void setup(){
size(300,300);
smooth();
background(255,255,255);
} void draw(){
fill(255,255,0);
ellipse(width/2,height/2,head,head);
fill(255,255,255);
//eyes
ellipse(width/2-40,height/2-30,eyes,eyes);
ellipse(width/2+40,height/2-30,eyes,eyes);
fill(0,0,0);
ellipse(width/2-40,height/2-30,10,10);
ellipse(width/2+40,height/2-30,10,10);
//mouth
fill(200,30,20);
ellipse(width/2,height/2+50,90,45);
}
</script>
